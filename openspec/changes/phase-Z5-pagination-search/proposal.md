# Proposal: Phase Z5-pagination-search — 工具与 Skills 分页搜索

## 背景

设置界面的工具管理和 Skills 管理当前为纯列表展示，当工具/Skills 数量较多时（当前已 30 个工具），缺乏：
- **搜索功能**：无法按名称或描述快速定位
- **分页功能**：所有项一次性渲染，未来随工具增多会影响性能和可用性

## 目标

- P1: 工具列表分页 — 每页默认 20 条，底部页码导航
- P2: 工具列表搜索 — 按名称 + 描述模糊搜索
- P3: Skills 列表分页 — 每页默认 20 条
- P4: Skills 列表搜索 — 按名称 + 描述 + body 内容模糊搜索

## 范围

| 变更 | 文件 | 影响 |
|------|------|------|
| 工具分页+搜索 | `Settings.tsx` | 中 |
| Skills 分页+搜索 | `Settings.tsx` | 中 |
| 分页/搜索组件样式 | `Settings.css` | 小 |
