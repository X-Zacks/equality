# Phase W: Skills UI 增强 — 提案

## 背景

当前 Skills 设置页为简单列表，展开详情以 `<pre>` 内联显示，且没有分类。
随着 Skill 数量增长（当前 20+），需要更好的浏览和分类体验。

## 目标

1. **Skill 详情抽屉**：点击 Skill 后在右侧弹出抽屉面板，展示完整 SKILL.md 内容
2. **Skill 分类**：按任务目的自动分类（开发、数据处理、文档处理、通信、工作流）
3. **分类筛选**：顶部分类 Tab，快速过滤

## 非目标

- 本阶段不做 Skill 在线安装/卸载 UI（Gallery 已有后端支持，前端集成放后续）
