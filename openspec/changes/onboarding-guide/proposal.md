# Proposal: 用户引导增强（Onboarding Guide）

## 背景

Equality 已具备 25 个子系统、28 内置工具、6 层 Skills、7 层安全等完整能力，但新用户首次启动时缺少系统性的引导流程。当前的 BOOTSTRAP.md 对话引导只关注"认识用户"（姓名/性格/偏好），未覆盖功能发现和使用教学。

## 目标

为初始用户提供**渐进式引导体验**，从首次启动到熟练使用：

1. **首次启动引导** — 模型配置 → 功能导览 → 第一次对话
2. **功能发现提示** — 在用户可能需要某功能时主动提示（如附件、@Skill、记忆等）
3. **示例场景库** — 内置可一键执行的示例对话，展示核心能力

## 范围

- 修改前端 Chat.tsx：新用户空状态 → 示例场景卡片
- 修改 BOOTSTRAP.md 模板：增加功能导览步骤
- 新增 bundled skill: `getting-started`
- 新增前端提示组件：功能发现提示

## 不做

- 不做视频教程
- 不做交互式 tutorial overlay（太重）
- 不改变核心架构
