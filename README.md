# Equality — AI 桌面智能助理

> 面向 Windows 用户的本地 AI Agent。真正理解你的工作上下文，能操作文件、执行代码、控制浏览器、搜索代码库，并在多 Agent 协作下完成复杂工程任务。

---

## 🎯 设计理念

Equality 不是聊天机器人，而是**可以持续进化的 AI 工作伙伴**。

它的核心思路是：**技能平权**——用强模型（如 GPT-5）完成一项复杂任务后，可以将完整的执行方法自动沉淀为"技能文档"。此后，成本更低的模型（如 DeepSeek V3）读取这份技能文档，同样能完成这项任务。随着技能库的积累，你使用 Equality 越久，它就越懂你的工作方式，同时 AI 调用成本也会不断降低。

---

## ✨ 核心能力

### 🤖 强大的 Agent 执行能力

Equality 的 Agent 能自主规划并完成多步骤任务，而不只是生成文字：

- **文件操作**：读写、编辑、搜索、批量处理本地文件，修改代码前自动展示差异供你确认
- **代码执行**：在沙箱环境中运行 Shell 命令，实时获取输出，自动处理错误重试
- **浏览器控制**：自动打开网页、点击、填表、截图，完成浏览器自动化任务
- **代码库搜索**：对整个项目进行语义级搜索，精确找到相关代码片段，而非简单关键词匹配
- **LSP 代码理解**：获取函数定义、查找引用、读取类型错误，真正理解代码结构
- **网络访问**：搜索网页、抓取内容，实时获取最新信息

### 🧠 长期记忆系统

Equality 拥有跨会话的持久化记忆，不会因为关闭窗口而遗忘：

- 你说"记住我偏好 TypeScript"，下次对话它会自动应用这个偏好
- 关闭应用再打开，所有历史对话和记忆完整保留
- 记忆基于语义检索，能理解"上次那个项目"指的是什么
- 支持在设置页面统一查看、编辑、导出所有记忆

### 📝 Skills 技能系统

Skills 是 Equality 的核心竞争力，让 AI 能力可以积累和复用：

- **自动沉淀**：复杂任务完成后，AI 会主动提议将执行方法保存为技能，下次同类任务直接调用
- **技能组合**：在输入框输入 `@` 可唤出技能选择器，支持多技能叠加使用
- **安全隔离**：每个技能文件经过安全扫描，防范恶意代码注入
- **灵活管理**：支持工作区级、个人级、系统级多层优先级，技能修改后即时生效

### 🎭 多 Agent 协作编排

面对复杂工程任务，Equality 可以调度多个专职 Agent 并行协作：

- **角色分工**：主管（Supervisor）→ 架构师 → 开发者 → 测试者 → 审查者，各司其职
- **DAG 任务图**：复杂任务拆解为有依赖关系的节点图，支持并发执行，最大化效率
- **实时监控**：可随时查看每个子 Agent 的状态，支持中途暂停、重定向指令、强制终止
- **深度安全**：子 Agent 嵌套深度受限，防止失控循环

### 🔐 多层安全保障

工具调用经过 7 层安全防护，防止 AI 产生破坏性操作：

- **策略管道**：可配置允许/拒绝哪类工具调用
- **变更分类**：自动识别读/写/执行操作，高风险操作需要明确授权
- **沙箱隔离**：Bash 命令在受限环境运行，防止路径穿越和符号链接攻击
- **注入防护**：过滤 14 种 Prompt Injection 模式，防止外部内容劫持 AI 行为
- **网络防护**：阻止 SSRF 攻击，防止 AI 访问内网敏感地址

## 🖥️ 桌面体验

- **流式对话**：AI 回复、工具调用过程实时可见，每个工具的输入输出均可展开查看
- **Diff 预览**：AI 写文件前展示行级差异，你来决定接受还是拒绝
- **交互式操作**：AI 可以在对话中插入按钮、下拉选择框，等待你的输入后继续执行
- **文件附件**：支持拖放文件、粘贴截图（图片、PDF、任意文本文件），最多 5 个
- **暂停与重定向**：执行中可暂停，输入新指令后 AI 会调整方向继续
- **会话管理**：多会话切换，按日期分组，父子 Agent 树形展示
- **费用追踪**：每次调用的 Token 消耗和估算费用实时记录
- **主题与缩放**：浅色/深色/跟随系统，界面可在 50%–200% 之间自由缩放
- **轻量安装**：~25MB 安装包，系统托盘常驻，关闭窗口不退出后台

## 🚀 快速开始

### 环境要求

Windows 系统，需安装：
- [Visual Studio Build Tools 2022](https://aka.ms/vs/17/release/vs_BuildTools.exe)（勾选"使用 C++ 的桌面开发"）
- Rust 工具链（`winget install Rustlang.Rustup`）
- Node.js ≥ 18 + pnpm ≥ 8

### 安装与运行

```bash
pnpm install
pnpm dev:core      # 终端 1：启动后端 Agent Core
pnpm dev:desktop   # 终端 2：启动桌面应用
```

### 第一次使用

1. 点击左侧 **⚙️ 设置** → **模型** Tab，配置至少一个 API Key
   - 最快：点击 GitHub Copilot 的「登录」按钮，完成浏览器授权（免费）
   - 推荐：同时配置 Copilot + DeepSeek，开启自动路由

2. 回到聊天页面，直接描述你想做的事，例如：
   - *"帮我分析一下 D:\projects\myapp 这个项目的结构"*
   - *"记住：我的所有项目都放在 D:\work 目录"*
   - *"@excel-cost-analysis"*（激活特定技能）

3. 探索更多用法：
   - 输入 `@` 唤出技能选择器，`#` 唤出工具选择器
   - 对话中说"每天早上 9 点提醒我看邮件"——AI 会创建定时任务
   - 设置 → **记忆** Tab，查看和管理所有跨会话记忆
